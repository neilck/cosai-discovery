# tests/

Pytest tests for the `cdx_indexer` Python package.

**Not to be confused with `_expected/`** — that directory holds per-project expectation files used for *eyeballing* the indexer's output. This directory holds *automated* unit/integration tests for the Python code itself.

Empty until Phase 1 work begins. See [`../_docs/indexer-build-plan.md`](../_docs/indexer-build-plan.md).
